Algorithm

Quick Sort is a Divide & Conquer sorting algorithm. It picks a pivot, partitions the array around it, then recursively sorts each side.

Divide

Select pivot, partition array around it

Conquer

Recursively sort left & right subarrays

Combine

Array is sorted in place — no merging needed

Recurrence Relation

T(n) = T(k) + T(n–k–1) + Θ(n)
k = elements left of pivot n–k–1 = elements right of pivot Θ(n) = partitioning cost

Each call partitions n elements in Θ(n) time, then recurses on two sub-problems whose sizes add up to n–1. In the best case the pivot splits the array evenly (k ≈ n/2), giving O(n log n). In the worst case one side is always empty, giving O(n²).
